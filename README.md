# Portfolio Website of Thomas, Freelance Graphic Designer

## Description
Portfolio website for Thomas, a freelance graphic designer, showcasing his visual identity, UI work, and illustrations.  
The site is responsive, fast, and faithfully follows the provided mockup for desktop and mobile (tablet version is free to adapt).  
The fixed menu allows navigation via anchors (#home, #articles, #portfolio, #contact).  
The "Download CV" button allows users to download a PDF document.

## Live Site Link
https://ddiaby.github.io/site-thomas/

## Technologies Used
- HTML5
- CSS3 (Flexbox / Grid)
- Fonts: Heebo via Google Fonts (or Poppins if Heebo is unavailable)
- Web-optimized images with descriptive alt attributes
- W3C validation compliant
- Basic accessibility: labels, lang="fr", ARIA landmarks if necessary

## Features
- Fixed main menu with anchor navigation
- Responsive design: desktop and mobile required, tablet version flexible but consistent
- Functional contact form with labels and alt attributes
- Button to download CV
- Layout faithfully follows the mockup (spacing, colors, typography)
- Optimized performance: compressed images, clean code
